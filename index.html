팝업 HTML 생성기

팝업 링크와 이미지 경로를 입력하면 기존 팝업 구조를 유지한 전체 코드를 자동으로 생성합니다. 링크 주소가 비어 있으면 자동으로 #none이 들어갑니다.

기존 팝업 구조 유지 전체 코드 한 번에 출력 새창 열기 접근성 반영 드래그 기능 제거

팝업 입력

팝업별 링크 주소, 이미지 경로, 새창 열기 여부를 입력해 주세요.

입력 후 코드 생성 버튼을 눌러주세요.

전체 코드 출력

HTML, CSS, JavaScript가 포함된 전체 코드를 한 번에 복사할 수 있습니다.

팝업 구조는 고정하고 링크, 이미지 경로, 새창 여부만 바꾸는 운영형 생성기입니다.

링크 주소가 비어 있으면 자동으로 #none이 출력됩니다.
